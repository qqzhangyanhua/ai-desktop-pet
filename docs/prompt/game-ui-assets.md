# Game UI Asset Prompts

Please generate the following images to match the "Cartoon Game UI" style.

## 1. Main Background
**Prompt:** A beautiful, bright fantasy nature landscape background for a game, blue sky with fluffy white clouds, green rolling hills, vibrant colors, cartoon style, slight blur effect for use as a background, high resolution.
**Filename:** `bg-nature.png`

## 2. Wooden Board Frame (Main Container)
**Prompt:** A large wooden game UI panel or board, empty center, cartoon style, rich wood texture, rounded corners, perhaps with some metal bolts or decorative corner pieces, slight drop shadow, isolated on white background.
**Filename:** `ui-board-wood.png`

## 3. Title Board (Header)
**Prompt:** A wooden signboard or banner for a game UI header, curved top, hanging or fixed on top of a panel, cartoon style, wood texture, empty space for text, isolated on white background.
**Filename:** `ui-title-board.png`

## 4. Parchment Panel (Info Area)
**Prompt:** A piece of old parchment paper or beige paper texture, rectangular with slightly rough edges, cartoon style, suitable for a text background in a game UI, isolated on white background.
**Filename:** `ui-parchment.png`

## 5. Glossy Buttons (Various Colors)
**Prompt:** A set of glossy, rounded rectangular game buttons in 3D cartoon style. 
- One **Blue** button.
- One **Orange** button.
- One **Green** button.
- One **Pink/Red** button.
- One **Purple** button.
- One **Brown** button.
Each button should look clickable, with a highlight and shadow. Isolated on white background.
**Filenames:** `btn-blue.png`, `btn-orange.png`, `btn-green.png`, `btn-pink.png`, `btn-purple.png`, `btn-brown.png`

## 6. Icons (Optional - can use Lucide, but custom is better)
**Prompt:** 3D cartoon icons for game UI:
- A Microphone (Voice).
- A Happy Face/Hand (Emotes).
- Two Photos/Portraits (Friends).
- A Gold Trophy (Achievements).
- A Bell (Notifications).
- A Shield (Privacy).
- A Fish (Feed).
- A Briefcase (Work).
- A Shopping Cart (Shop).
- A House (Home).
**Filenames:** `icon-mic.png`, `icon-emote.png`, `icon-friends.png`, `icon-trophy.png`, `icon-bell.png`, `icon-shield.png`, `icon-feed.png`, `icon-work.png`, `icon-shop.png`, `icon-home.png`
