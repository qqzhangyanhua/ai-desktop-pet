# 📱 在桌面应用中查看 Live2D 日志

## 🎯 我已经为您添加了内置调试面板！

### 第一步：重新打包应用

```bash
cd /Users/zhangyanhua/AI/desk/ai-desktop-pet
pnpm build:tauri
```

### 第二步：安装新版本

打包完成后，安装新的应用：
```bash
open "src-tauri/target/release/bundle/macos/AI Desktop Pet.app"
```

### 第三步：打开调试面板

在应用运行时，有**两种方式**打开调试面板：

#### 方式 1：点击按钮（推荐）⭐
- 在应用窗口的**右下角**会有一个 **🐛 调试** 按钮
- 点击这个按钮就会打开调试面板

#### 方式 2：使用快捷键
- 按 **Command + D** (macOS)
- 或 **Ctrl + D** (Windows/Linux)

### 第四步：查看日志

调试面板会显示：
- ✅ 所有 Live2D 相关的日志
- ✅ 时间戳
- ✅ 错误会用红色高亮显示
- ✅ 警告会用黄色高亮显示

### 第五步：发送日志给我

在调试面板中：
1. 点击 **"复制全部"** 按钮
2. 日志会自动复制到剪贴板
3. 粘贴发送给我

或者直接截图发送！

## 📸 调试面板功能

- **清空**：清除所有日志
- **复制全部**：复制所有日志到剪贴板
- **关闭**：关闭调试面板（按 Cmd+D 可重新打开）

## 🔍 需要查找的关键信息

### ✅ 成功的日志应该包含：
```
[GlobalLive2D] ✓ 成功! Content-Type: application/json
[GlobalLive2D] ✅ 找到可用路径
```

### ❌ 失败的日志可能包含：
```
[GlobalLive2D] ✗ 失败: 404 Not Found
[GlobalLive2D] ✗ 错误: Failed to fetch
[GlobalLive2D] ❌ 所有路径尝试均失败！
```

## 💡 提示

- 调试按钮在应用启动后就会出现在右下角
- 如果看不到按钮，尝试按 **Cmd+D** 快捷键
- 调试面板会自动记录所有 Live2D 相关的日志
- 最多保留最近 50 条日志
